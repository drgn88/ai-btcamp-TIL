# B7
## Problem
---
입력받은 4자리 이상 정수에서 100원 미만을 반올림 하는 코드를 구현하시오
즉, 50원 이상이면 올림처리하고 49원 이하이면 내림처리를 한다
예를들어, 1249원이면 100원 미만이 49원이므로 1200원이 된다. 4350원이면 100원 미만이 50원이므로 4400원이 된다
## Input Example
---
4자리 이상의 정수 입력
1249
## Output Example
---
100원 미만을 반올림 처리한 결과를 출력
1200
