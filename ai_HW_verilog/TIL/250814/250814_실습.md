# R Type RV32I

<img src="./img/스크린샷 2025-08-14 112307.png"><br>

- Register Source, Destination
  - Register File의 Address(5 bit)
- Opcode
  - Type에 따라 다름
  - 명령어 타입을 결정함
- Function
  - 연산을 구분
  - ADD인지, SUB인지 등등

EX) ADD: rd = rs1 + rs2
- rd: wr_addr
- rs1: r_addr_1
- rs2: r_addr_2

EX) Assembly: ADD X7, X5, X6
- ADD X7 = X5 + X6
- X7: wr_addr
- X5: r_addr_1
- X6: r_addr_2
